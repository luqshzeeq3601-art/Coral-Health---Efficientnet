# Output Folder Guide

This folder is organized to avoid mixing baseline-only results with the
baseline-vs-current-model comparison package.

## `baseline_model/`

Contains outputs from the baseline EfficientNetB0 model only.

Use these files when explaining the baseline model by itself:

- `classification_report.txt`
- `classification_report.png`
- `confusion_matrix.png`
- `eval_summary.json`
- `gradcam_outputs.png`
- `model_summary.txt`
- `training_history.json`
- `training_history.png`

Generated by:

```powershell
python 05_Baseline_Model\train_baseline.py
```

## `comparison/`

Contains outputs comparing the baseline model against the current V4 Robust
EfficientNetB0 model.

Use these files for thesis and presentation comparison:

- `01_overall_performance_comparison.png`
- `02_per_class_f1_comparison.png`
- `03_confusion_matrix_pair.png`
- `04_training_curves_comparison.png`
- `05_improvement_delta_chart.png`
- `06_error_reduction_by_class.png`
- `07_gradcam_qualitative_panel.png`
- `08_complexity_cost_comparison.png`
- `09_comparison_summary_table.png`
- `10_method_rationale_matrix.png`
- `11_ablation_plan_table.png`
- `comparison_summary.txt`
- `comparison_talking_points.md`
- `thesis_slide_comparison_guide.md`

Generated by:

```powershell
python 05_Baseline_Model\compare_models.py
```
